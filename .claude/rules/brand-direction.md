# Brand Direction (CRITICAL)

## 브랜드 정의

**MUTON = MUTE + TONE**

건축적 음영과 감각의 조화를 추구하는 하이엔드 주방가구 브랜드

### 브랜드 약속

> Precision that completes architecture.
> 정밀함으로 건축을 완성하다.

---

## 톤 키워드

```
Mute / Tone / Shadow / Precision / Sentence / Temperature
```

---

## 핵심 가치

| 가치 | 의미 |
|------|------|
| 절제 | 불필요한 요소 제거 |
| 정밀 | 이음, 단차, 경계, 그림자까지 설계 |
| 온도 | 차가운 도시성 + 따뜻한 촉감의 균형 |
| 증거 | 말보다 결과물 |

---

## 브랜드 보이스

- 말 수는 적다
- 문장은 짧다
- 감각적이되 과장하지 않는다
- "럭셔리"를 외치지 않고 증명한다

---

## 카피 가이드

### 금지 표현

```
최고급, 프리미엄, 명품, 럭셔리, 하이엔드, 최상의, 완벽한
```

### 권장 표현

```
재료, 빛, 결, 정밀, 조립, 경계, 온도, 음영, 질감, 단차, 이음
```

### 예시

```
❌ "최고급 프리미엄 주방가구"
✅ "경계를 설계합니다"

❌ "완벽한 마감으로 최상의 품질"
✅ "1mm 단차가 공간의 문장을 완성합니다"
```
