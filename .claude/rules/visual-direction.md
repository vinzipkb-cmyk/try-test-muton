# Visual Direction (CRITICAL)

## 시각 원칙

```
Shadow-led Minimalism + Material Truth + Editorial Grid
```

---

## 컬러

| 구분 | 값 |
|------|-----|
| 기본 | 모노톤 (오프화이트 / 차콜) |
| 포인트 | 1개만 (브랜드 악센트) |
| 라이트 모드 | 재료의 결을 드러냄 |
| 다크 모드 | 형태의 윤곽과 그림자를 드러냄 |

※ 다크/라이트는 "모드 전환"이 아닌 **조도 변화**

---

## 레이아웃

| 요소 | 값 |
|------|-----|
| Border Radius | 0 (각진 형태) |
| Border | 1px |
| Grid Gap | 1px (Archive) |
| Spacing | 4/8px 기반 |

---

## 타이포그래피

| 구분 | 스타일 |
|------|--------|
| 제목 | 짧고 강하게 |
| 본문 | 얇고 길게 |

---

## 이미지/영상

| 요소 | 지침 |
|------|------|
| 카메라 | 과한 광각 금지, 재료/디테일 중심 |
| 빛 | 하이라이트보다 그림자/그라데이션 |
| 질감 | 목재 결, 석재 표면, 금속 브러시 |
| 배경 | 정보 소음 제거, 클린 |

---

## 모션

### 허용

| 패턴 | 속도 |
|------|------|
| Fade In | 0.3s ease-out |
| Hover Overlay | 0.2s ease |
| Scroll Fade | viewport 기반 |
| Page Transition | 0.2s fade |

### 금지

- Bounce, Spring
- 과도한 Parallax
- Glow, Pulse
- 자동재생 캐로셀

---

## 체크리스트

### 비주얼
- [ ] 과한 광각 없음
- [ ] 하이라이트 과다 노출 없음
- [ ] 배경 소음 없음

### UI
- [ ] 글로우/바운스 없음
- [ ] 과도한 패럴랙스 없음
- [ ] border-radius: 0

### 콘텐츠
- [ ] 자기선언형 수식어 없음
- [ ] 이모지 없음
